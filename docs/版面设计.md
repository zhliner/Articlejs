# 版面设计

## 基本因素

- 大小。（形状，整体）
- 颜色。（内容，背景）
- 区隔。（空白，非空白）


### 大小

- 字号。
- 色块。


### 颜色

- 纯色。
- 相似色。
- 混色。


### 区隔

- 线条。
- 区块。
- 间距。
- 缩进。


## 内容影响

- 形状。如：字体。
- 结构。如：自身版式（小）。


## 时间维

- 交互。（简洁，合乎逻辑）
- 特效。（意蕴相关，适可而止）
- 动画。（另一种内容）
- 听觉。（辅助，提示）


## 目标

错落有致，易于阅读（识别率/眼）。


## 注记

- CSS：结构和表现应当分离。grid/flex。
- HTML：行为和内容应当紧邻。Tpb:On/By/To。
