# $.empty( el ): Element

清空 `el` 元素的内容，包括子元素、文本节点和注释节点等任意子节点。


## 参数

### el: Element

清空其内容的目标元素。


### 返回值

被清空的目标元素自身。


## 示例

```html
<p id="test">
    <em>Internet is</em> a <strong>World</strong>!
</p>
```

### 单元素版

```js
let em = $.get('#test em');  // <em>Internet is</em>

$.empty(em);
// <p id="test">
//     <em></em> a <strong>World</strong>!
// </p>
```


### 集合版

```js
$('#test > *').empty();
// <p id="test">
//     <em></em> a <strong></strong>!
// </p>

$('#test').empty();
// <p id="test"></p>
```
