# $.isXML( el ): boolean

检查目标是否为一个 XML 节点。


## 参数

### el: {Element|Object}

元素节点或一个对象。


## 示例

```html
<svg baseProfile="full" width="300" height="200">
    <rect width="50%" height="100%" fill="orangered"></rect>
    <circle cx="150" cy="100" r="80" fill="green"></circle>
</svg>
```

```js
r = $.get('rect');
$.isXML(r)  // true

$.isXML(document.body)  // false
```
