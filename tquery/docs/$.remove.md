# $.detach( node ): this

删除文档树中的节点（元素或文本节点），返回调用者自身（`$`）。

> **注**：效果类似于未保存返回值的 `$.detach` 接口。


## 参数

### node: Node

要被删除的DOM树中的节点。


## 示例

```html
<p id="test">
    <em>Internet is</em> the <strong>World</strong>!
</p>
```

### 单元素版

```js
let em = $.get('#test em');  // <em>Internet is</em>

$.remove(em);
// <p id="test">
//      the <strong>World</strong>!
// </p>
```


### 集合版

```js
$('#test > *').remove();
// <p id="test">
//      the !
// </p>
```
