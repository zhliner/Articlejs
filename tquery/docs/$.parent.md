# $.parent( el, slr ): Element | null

获取 `el` 的直接父元素。`slr` 为选择器或测试函数，用于测试父元素是否匹配。


## 参数

### el: {Element}

检索其父元素的起点元素。


### slr: {String|Function}

测试父元素是否匹配的选择器或测试函数。


## 示例

```html
<p id="test">
    这是一个<strong>通用的</strong><em>测试段落</em>。包含了一些<em>行内元素</em>。
</p>
```

```js
let el = $.get('#test > em');  // <em>测试段落</em>

$.parent(el);                  // <p id="test">...</p>
$.parent(el, '#abc');          // null
$.parent(el, e => true);       // <p id="test">...</p>
```
