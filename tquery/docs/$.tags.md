# $.tags( code ): String

将伪标签源码（由 `[]` 包围标签）转化为正常的HTML源码，可用于元素属性值中直观地包含标签源码。

实现只是一种简单的替换，如果代码中本来就需要包含 `[]`，可以前置字符 `\` 转义，避免被 `<>` 替换。


## 参数

### code: String

用 `[]` 包围标签的伪 HTML 源码。


## 示例

```html
<a href="#" title="[a] is a \[link\]">Click me</a>
```

```js
let a = $.get('a'),             // <a>
    ttl = $.attr(a, 'title');   // "[a] is a \[link\]"

let t2 = $.tags(ttl);
// "<a> is a [link]"

$.attr(a, 'title', t2);
// <a href="#" title="<a> is a [link]">Click me</a>
```
