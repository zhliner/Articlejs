# $.parents( el, slr ): [Element]

获取 `el` 元素的上级元素集。`slr` 为可选的选择器或测试函数，用于过滤。


## 参数

### el: {Element}

检索其上级元素的起点元素。


### slr: {String|Function}

测试上级元素是否匹配的选择器或测试函数，匹配的就加入结果集。



## 示例

```html
<html>
    <head></head>
    <body class="Body">
        <p id="test" class="Test">
            这是一个<strong>通用的</strong><em>测试段落</em>。包含了一些<em>行内元素</em>。
        </p>
    </body>
</html>
```

```js
let el = $.get('#test > em');  // <em>测试段落</em>

$.parents(el);
// [
//     <p>...</p>,
//     <body>...</body>,
//     <html>...</html>
// ]

$.parents(el, '[class]');
// [
//     <p>...</p>,
//     <body>...</body>
// ]

$.parents(el, (e, i) => i == 2);
// [
//     <body>...</body>
// ]
```
