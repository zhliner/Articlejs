# $.create( html, exclude, doc? ): DocumentFragment

创建文档片段，片段中不包含 `<script>`、`<style>`、`<link>` 三种元素。


## 参数

### html: {String}

文档片段的源码内容，仅支持字符串实参。源码应当为规范的HTML结构，可以赋值给 `<div>` 元素的 `innerHTML` 属性。


### exclude: {Array}

考虑安全性，`<script>`、`<style>`、`<link>` 这3中元素会被强制清理。本参数提供一个存储空间，被清理的三种元素会存放到此。
